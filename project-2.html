Project 2

Welcome to my Personal Dashboard! This project showcases live updates on cryptocurrency prices, local weather, and a dynamic background image, all in one visually appealing interface.

Live Link

Project Overview

This Personal Dashboard integrates live data from various sources to create an interactive and dynamic user experience. It features: Cryptocurrency Prices: Displays the current price, 24-hour high, and low of Dogecoin using data from the CoinGecko API. Local Weather: Shows the current temperature, weather conditions, and location using the OpenWeatherMap API based on the user's geolocation. Dynamic Background: Fetches and sets a beautiful random nature background image from the Unsplash API. Real-Time Clock: Updates every second to show the current time. Error Handling: Gracefully handles API errors and provides fallback content to ensure a smooth user experience.

Technology Used Frontend: HTML: For structuring the content of the web page.
CSS: For styling and layout, ensuring the dashboard is visually appealing and responsive.
JavaScript: For fetching data from APIs, updating the DOM, and handling user interactions. APIs:
Unsplash API: For fetching random nature background images.
CoinGecko API: For retrieving the latest Dogecoin prices and related data.
OpenWeatherMap API: For obtaining local weather information based on user geolocation. Geolocation API: For accessing the user's current location to provide localized weather data.
